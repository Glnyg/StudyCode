# 07-Platform 阅读导航

## 这部分讲什么
- 这里讲系统的运行平台、集群分池、观测、交付和生产部署基线。

## 为什么重要
- 平台基线决定服务最终跑在哪里、怎么隔离、怎么回滚和怎么观测。
- 如果平台设计只停留在“有 [[Kubernetes]] 集群”，真正的生产约束就会缺位。
